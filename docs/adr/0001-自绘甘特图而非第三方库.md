# 0001-自绘甘特图而非第三方库

## 状态

已接受

## 背景

交付计划模块需要甘特图展示 SOP 步骤的时间线。Arco Design 没有内置甘特图组件，需选择实现方案。

## 决策

采用纯 CSS + React 自绘甘特图，不引入第三方甘特图库（如 frappe-gantt、dhtmlx-gantt）或图表库（如 @antv/g2plot）。

## 理由

1. 第一版甘特图为纯展示型（无拖拽），自绘完全满足需求
2. 避免引入新依赖，保持项目轻量
3. 样式完全可控，和 Arco + Tailwind 风格统一
4. 左右分栏（任务列表 + 时间轴）本身就是两个 div，同步滚动用 onScroll 联动

## 后果

- 优势：零新增依赖，风格统一，后续可逐步增强交互
- 风险：若后续需要拖拽排期、依赖连线等高级功能，自绘方案的工作量会显著增加，可能需要重新评估引入第三方库
